## Capability

Reviews content for SEO performance and suggests optimisations: keyword placement, meta tags, heading structure, internal linking opportunities, and readability.

## When to Use

- Before publishing any web content
- When updating existing content for better search ranking
- After drafting blog posts, landing pages, or product descriptions

## What It Does

1. **Keyword analysis** — density, placement, natural integration
2. **Meta tag generation** — title tag, meta description, Open Graph tags
3. **Heading structure** — H1-H6 hierarchy for crawlers
4. **Readability scoring** — sentence length, paragraph structure
5. **Internal linking** — opportunities to link to related content
6. **Technical SEO** — image alt text, URL structure suggestions

## Inputs

Content to optimise and target keywords.

## Outputs

Optimised content with SEO recommendations report.
