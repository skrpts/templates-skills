## Capability

Analyses structured data — CSV, JSON, or tabular data — to identify patterns, trends, outliers, and actionable insights.

## When to Use

- After importing a dataset that needs exploration
- When you need summary statistics, distributions, or trend analysis
- As a first step before more targeted analysis

## What It Does

1. **Data profiling** — column types, missing values, unique counts, distributions
2. **Summary statistics** — mean, median, mode, standard deviation, percentiles
3. **Trend identification** — time-series patterns, growth rates, seasonality
4. **Outlier detection** — values that fall outside expected ranges
5. **Correlation analysis** — relationships between variables
6. **Insight generation** — actionable observations from the data

## Inputs

The dataset to analyse (CSV, JSON, or tabular text).

## Outputs

Structured analysis report with statistics, visualisation descriptions, and insights.
